# TLNT — рекламный кит (Google + Instagram/Meta)

Бюджет: **100 AED/день (~3000/мес)**. Сплит на старт: **Google Search 60 · Instagram/Meta 40**. Telegram-посевы добавляем на 2–3-й неделе по результатам.

Цель: заявки в WhatsApp/форму. Считаем стоимость заявки (CPL) → переливаем бюджет в то, что дешевле.

## ЧТО НАСТРОИТЬ ДО ЗАПУСКА (только ты, без терминала)
1. **Instagram → Бизнес-аккаунт** (Настройки → Тип аккаунта). Заполнить: имя «TLNT — подбор персонала в ОАЭ», био, ссылка `tlnt.ae`, аватар-логотип.
2. **Facebook-страница** + связать с Instagram (без неё реклама не идёт).
3. **Meta Business Suite → Ads Manager** → создать рекламный аккаунт, привязать карту.
4. **Meta Pixel** (Events Manager) → прислать мне Pixel ID, я поставлю на сайт (ретаргетинг + конверсии).
5. **Google Ads** аккаунт → привязать карту.
6. (Позже) **Telegram-канал** для посевов.
